프로그래머스 - 기둥과 보 설치

18 Sep 2021 Algorithm C++ Online judge 구현/시뮬레이션 Programmers



Problem

2020 KAKAO BLIND RECRUITMENT 기둥과 보 설치

문제는 2차원 공간에 구조물 (기둥 or 보)를 설치/삭제 한 후 최종 구조물의 상태를 return 하는 것이다. 구조물의 정보(x 좌표, y좌표, 구조물 종류, 설치/삭제)들을 담은 2차원 배열이 입력되고, 최종 구조물의 상태(x 좌표, y 좌표, 구조물 종류)를 반환하면 된다. 단, 문제에 나온 특정 조건을 만족해야만 주어진 위치에 기둥과 보를 설치/삭제할 수 있다는 점에 유의해야 한다.


Solution

보와 기둥의 존재 여부를 나타내는 2차원 배열을 각각 만들고, 현재 특정 위치에 구조물이 존재하는 지를 표시한다. 구현할 때, 구조물 설치/삭제 입력에 대해 설치와 삭제 시에 만족해야 하는 특정 조건을 검사하는 것이 까다롭다. 기둥과 보가 존재하기 위해서는 각각의 조건을 만족해야 한다.

  • 기둥
    • 바닥 위에 존재
    • 왼쪽/오른쪽 보의 끝 위에 존재
    • 다른 기둥 위에 존재
  • 보
    • 왼쪽/오른쪽 끝 부분이 기둥 위에 존재
    • 양쪽 끝 부분이 다른 보와 동시에 연결

설치 시에는 위의 조건을 만족하는 지 확인하고, 만족할 경우 설치를 해주면 된다. 반면 삭제할 때는 삭제하려는 기둥, 보의 존재 여부에 영향을 받는 기둥, 보에 대해 위의 유효 조건을 검사해야 한다. 조건을 만족하지 않을 경우 삭제를 할 수 없으므로, 구조물 삭제를 취소해야 한다. x,y 좌표에 있는 기둥 또는 보 삭제 시 영향을 받는 구조물은 다음과 같다.

  • 기둥 (x, y)
    • 바로 위에 존재하는 기둥 (x, y+1)
    • 오른쪽 끝이 기둥 위에 존재하는 보 (x-1, y+1)
    • 왼쪽 끝이 기둥 위에 존재하는 보 (x, y+1)
  • 보 (x, y)
    • 왼쪽 끝 위에 존재하는 기둥 (x, y)
    • 오른쪽 끝 위에 존재하는 기둥 (x+1, y)
    • 왼쪽 끝에 닿아있는 보 (x-1, y)
    • 오른쪽 끝에 닿아있는 보 (x+1, y)

해당 위치에 있는 특정 기둥과 보만을 검사하면, 모든 구조물을 검사하지 않고 삭제가 가능한지 효과적으로 판단할 수 있다. 주의해야 할 점은, 검사해야 하는 위치에 구조물이 있을 경우에만 유효성 검사를 해야 하는 것이다. 이점을 신경쓰지 못해서 모든 케이스를 통과하는 데 많은 시간이 걸렸다.


Code

#include <vector>
using namespace std;

bool map[2][101][101];

// 기둥 놓을 수 있는지 확인
bool check_0(int n, int x, int y){
    if ((y == 0) ||
        (x > 0 && map[1][y][x-1]) ||
        (map[1][y][x]) ||
        (y > 0 && map[0][y-1][x])) {return true;}
    else {return false;}
}

// 보 놓을 수 있는지 확인
bool check_1(int n, int x, int y){
    if ((y > 0 && map[0][y-1][x]) ||
        (x < n && y > 0 && map[0][y-1][x+1]) ||
        (x > 0 && x < n && map[1][y][x-1] && map[1][y][x+1])){return true;}
    else {return false;}
}

// 기둥 지울 수 있는지 확인
bool check_d_0 (int n, int x, int y){
    if ((!map[0][y+1][x] || check_0(n, x, y + 1)) &&
        (!map[1][y+1][x] || check_1(n, x, y + 1)) &&
        (!map[1][y+1][x-1] || check_1(n, x-1, y + 1))){
        return true;
    }else{
        return false;
    }
}

// 보 지울 수 있는지 확인 
bool check_d_1(int n, int x, int y){
    if ((!map[1][y][x-1] || check_1(n, x - 1, y)) &&
        (!map[1][y][x+1] || check_1(n, x + 1, y)) &&
        (!map[0][y][x] || check_0(n, x, y)) &&
        (!map[0][y][x+1] || check_0(n, x + 1, y))){
        return true;
    }else {
        return false;
    }
}

void construct(int n, int x, int y, int a){
    if (a == 0 && check_0(n, x, y)){
        map[0][y][x] = true;
    }else if(a == 1 && check_1(n, x, y)) {
        map[1][y][x] = true;
    }
}

void deconstruct(int n, int x, int y, int a){
    if (a == 0){
        map[0][y][x] = false;
        if (!check_d_0(n, x ,y)) map[0][y][x] = true;

    }else if (a == 1){
        map[1][y][x] = false;
        if (!check_d_1(n, x, y)) map[1][y][x] = true;
    }
}

vector<vector<int>> solution(int n, vector<vector<int>> build_frame) {
    vector<vector<int>> answer;
    
    for (int i = 0; i < build_frame.size(); i++){
        int x = build_frame[i][0];
        int y = build_frame[i][1];
        int a = build_frame[i][2];
        int b = build_frame[i][3];
        
        if (b == 1){
            construct(n, x, y, a);
        }else {
            deconstruct(n, x, y, a);
        }
    }
    
    for (int i = 0; i <= n; i++){
        for (int j = 0; j <= n; j++){
            vector<int> each_0 (3);
            vector<int> each_1 (3);
            
            if (map[0][j][i]) {
                each_0[0] = i;
                each_0[1] = j;
                each_0[2] = 0;
                answer.push_back(each_0);
            }
            
            if (map[1][j][i]){
                each_1[0] = i;
                each_1[1] = j;
                each_1[2] = 1;
                answer.push_back(each_1);
            }
        }
    }
    
    return answer;
}
