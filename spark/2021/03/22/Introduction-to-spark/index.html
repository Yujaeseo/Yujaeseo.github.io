Apache Spark란

Apache Spark는 클러스터 환경에서 데이터를 병렬로 처리하는 컴퓨팅 엔진이자 라이브러리의 집합이다. 분산처리시스템인 Spark는 하나의 서버에서 처리할 수 있는 수준 이상의 대용량 데이터 처리를 빠르고 쉽게 할 수 있도록 도와준다. 즉 대용량 데이터를 처리할 때 클러스터 내에 있는 컴퓨터들 간 작업을 관리하고 조율하는 역할을 한다. Spark는 데이터 저장을 위한 솔루션이라기 보단 효과적인 데이터 처리를 목적으로 만들어졌다. 또한 비슷한 목적으로 개발된 Apache MapReduce와는 달리 반드시 Apache Hadoop과 실행해야 되는 것은 아니다. Python, Java, Scala, R 네 가지 언어를 지원하고 스트리밍, 머신러닝 등 다양한 라이브러리를 제공한다.


기본 아키텍처

Spark는 driver process와 다수의 executor 프로세스로 구성된다. Driver process는 클러스터 노드 중 하나에서 실행되며 main() 함수를 실행한다. 특정 작업을 병렬적으로 수행하기 위해 driver process는 다수의 executor에게 작업을 할당하고 executor는 할당받은 작업을 수행하게 된다. 사용자는 클러스터의 각 노드에 할당할 executor의 수를 지정할 수 있다. Spark가 연산에 사용할 클러스터는 Spark standalone 클러스터 매니저, Hadoop YARN, Mesos 같은 클러스터 매니저에서 관리한다. 사용자는 이 클러스터 매니저에 application을 제출하게 되고, 클러스터 매니저는 application 실행에 필요한 자원을 할당하여 사용자가 작업을 처리할 수 있도록 한다.
