Deploying Machine Learning Models in Production_Model Servers-Model Servers_Other Providers

09 Aug 2022 MLops Deep learning Machine learning Lecture summary

  1. NVIDIA Triton Inference Server

    • Triton Inference Server는 NVIDIA에서 제공하는 model server이다.

    • 오픈소스 inference 서빙 플랫폼으로서 tensorflow, pytorch, ONNX 등 타 딥러닝 프레임워크에서 학습된 모델들도 deploy를 할수 있다.

  2. Architecture

    • 서로 다른 framework에서 만들어진 모델을 single GPU에서 concurrently하게 실행할 수 있다.
    • Multi GPU 환경에서는 각 모델에 대한 instance가 생성되며 각 GPU에 할당된다.
      • User의 별도 coding 없이 GPU utilization을 높일 수 있다.
    • Batch inference와 streaming input에 대한 inference도 지원한다.
    • Performance를 높이기 위해 inference input, output을 CUDA GPU shared memory에 적재할 수도 있다.
      • httpc, grpc overhead를 줄일 수 있다.

  3. Designed for Scalability

    • Autoscaling, orchestration을 위해 K8s와 결합되었고, end-to-end ML workflow를 위해 Kubeflow와도 결합된다.
    • Http, grpc API를 지원하며 이를 통해 load balancer 통신할 수 있다.
    • 수많은 모델들이 동시에 serving될 수 있으며, GPU 뿐만 아니라 model을 heterogeneous한 환경에서 돌릴 수 있다.
      • Peak load 시에는 여러 프로세서 자원에 load를 분산시켜 효율적으로 대응할 수 있다.

  4. Torch Serve

    • Torchserve는 AWS와 Facebook이 함께 만들었으며, Pytorch model을 위한 serving framework이다.
    • 여러 기능을 제공하며, 무엇보다 opensource여서 자신에게 fit하게 customize할 수 있다는 장점이 있다.

  5. TorchServe Architecture (1)

    • Torchserve의 아키텍쳐는 다음과 같습니다.

  6. TorchServe Architecture (2)

    • Frontend는 request와 response 처리와 model lifecycle을 관리한다.

  7. TorchServe Architecture (3)

    • Backend는 model store로 부터 load된 model instance를 실행하며, inference를 담당한다.
    • 여러 model worker가 실행중인 것을 볼 수 있으며, 동일 model에 대한 여러 worker , 또는 서로 다른 model에 대한 여러 worker가 실행될 수 있다.

  8. TorchServe Architecture (4)

    • 모델은 cloud 또는 local storage로 부터 읽을 수 있다.

  9. TorchServe Architecture (5)

    • Server는 management, inference를 위한 API 또한 제공한다.

  10. KFServing

    • Kubeflow 또난 KF serving을 통한 serving 서비스를 제공하며, 여러 framework에서 만들어진 모델 serving을 지원한다.
